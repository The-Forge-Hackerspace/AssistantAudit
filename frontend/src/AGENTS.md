# FRONTEND SOURCE KNOWLEDGE BASE

**Path:** `/frontend/src/`
**Target:** Next.js 16 App Router (React 19 + TypeScript)

## OVERVIEW
Main application logic and UI. Follows App Router conventions with a clear separation between data fetching (hooks/services) and UI components.

## STRUCTURE
```
src/
├── app/            # App Router pages and layouts
├── components/     # UI components (has its own AGENTS.md)
│   └── ui/         # shadcn/ui primitives (DO NOT MODIFY)
├── contexts/       # React Contexts (auth-context.tsx)
├── hooks/          # Custom hooks (use-api.ts, use-mobile.ts)
├── lib/            # Utilities (api-client.ts, constants.ts, utils.ts)
├── services/       # API service layer (api.ts)
└── types/          # TypeScript definitions (api.ts, index.ts)
```

## APP ROUTER PAGES
| Route | Purpose |
|-------|---------|
| `/` | Dashboard with audit statistics |
| `/login/` | Authentication (email/password) |
| `/audits/` | Audit management and results |
| `/entreprises/` | Client/company management |
| `/sites/` | Physical site inventory |
| `/equipements/` | Network and IT equipment list |
| `/frameworks/` | Compliance referential viewer |
| `/outils/` | Interface for integrated audit tools |
| `/profile/` | User settings and credentials |

## WHERE TO LOOK
- **Add a page**: Create a folder in `app/` with `page.tsx`
- **Modify API logic**: Update `services/api.ts` or `lib/api-client.ts`
- **Add a type**: Update `types/api.ts` or `types/index.ts`
- **Modify Auth**: Check `contexts/auth-context.tsx` and `components/auth-guard.tsx`

## DATA FLOW
1. **Page/Component**: Calls a service function or `useApi` hook.
2. **useApi / Services**: Wraps `api-client` calls, managing SWR state or direct async requests.
3. **api-client**: Axios instance with JWT interceptors for automatic auth headers.
4. **Backend**: FastAPI endpoints (see `/backend/app/api/v1/`).

## AUTH FLOW
- **AuthContext**: Holds global user state and token management logic.
- **AuthGuard**: Wrapper component in `app/layout.tsx` that redirects unauthenticated users to `/login`.
- **Interceptors**: `lib/api-client.ts` automatically attaches the JWT and handles 401 token refresh.

## CONVENTIONS
- **Path Aliases**: Use `@/` to refer to the `src/` directory.
- **Language**: UI text and user-facing strings are in **French**. Code, variables, and comments are in **English**.
- **State Management**: Prefer SWR for server state. Use React Context only for global UI/auth state.
- **Components**: UI primitives in `components/ui/` are generated by shadcn/ui. Do not edit them directly.
