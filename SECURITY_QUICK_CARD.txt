  ⚠️  MEDIUM: Hardcoded Temporary File Path (Line 490)

      Current Code:
      ps1_path = Path("D:/AssistantAudit/temp/monkey365_scan.ps1")

      Issues:
      • Predictable file location
      • Race conditions possible
      • Not portable

      Fix (30 minutes):
      ┌──────────────────────────────────────────────────────────────────┐
      │ import tempfile, stat, os                                        │
      │                                                                  │
      │ temp_dir = Path(tempfile.gettempdir()) / "assistantaudit_m365"  │
      │ temp_dir.mkdir(parents=True, exist_ok=True, mode=0o700)         │
      │ ps1_path = temp_dir / f"scan_{scan_id}_{os.getpid()}.ps1"       │
      │ ps1_path.write_text(script, encoding="utf-8")                   │
      │ os.chmod(ps1_path, stat.S_IRUSR | stat.S_IWUSR)  # 0o600        │
      └──────────────────────────────────────────────────────────────────┘

      See: SECURITY_FIXES_MONKEY365.py (line 20-80)
